
# Blueprint

## Overview

This application is designed to help teachers manage their students and their progress. It allows teachers to create and manage student profiles, track their progress, and create and manage learning activities.

## Features

### Implemented

*   **Student Management:** Create, read, update, and delete student profiles.
*   **Level Management:** Create, read, update, and delete levels.
*   **List Management:** Create, read, update, and delete lists of words.
*   **History:** Track student progress.
*   **Tabbed Student View:** Students are organized by level in a tabbed interface on the home page.
*   **Settings Page:** A dedicated page for application settings.
*   **Theme Selection:** Users can switch between light, dark, and system themes from the settings page.
*   **Admin Page:** A separate page for all administrative tasks, accessible from the settings page.
*   **Take Photo:** Users can take photos to use as images for lists and words.
